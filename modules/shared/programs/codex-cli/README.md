# Codex CLI

OpenAI's [Codex CLI](https://developers.openai.com/codex/cli) (`codex`).

## 🤹 Installing agent skills

Codex CLI does not have commands for installing agent skills. Use the `skills` CLI ([vercel-labs/skills](https://www.npmjs.com/package/skills)):

```bash
$ skills add vercel-labs/agent-skills [--skill <skill-name>]
```

## 🛠️ Setting up MCP servers

Check `codex mcp add --help` command.
