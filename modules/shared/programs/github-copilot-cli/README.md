# GitHub Copilot CLI

GitHub's [Copilot CLI](https://github.com/github/copilot-cli) (`copilot`).

## 🤹 Installing agent skills

Use the `skills` CLI ([vercel-labs/skills](https://www.npmjs.com/package/skills)). Or you prefer, You can use GitHub CLI (`gh`) to install agent skills:

```bash
$ gh skill install github/awesome-copilot documentation-writer
```

## 🛠️ Setting up MCP servers

Check `copilot mcp add --help` command.
