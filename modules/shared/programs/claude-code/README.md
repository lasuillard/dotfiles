# Claude Code

Anthropic's [Claude Code](https://code.claude.com/docs/en/overview) (`claude`).

## 🤹 Installing agent skills

Claude Code does not have commands for installing agent skills. Use the `skills` CLI ([vercel-labs/skills](https://www.npmjs.com/package/skills)):

```bash
$ skills add vercel-labs/agent-skills [--skill <skill-name>] --agent claude-code
```

## 🛠️ Setting up MCP servers

Check `claude mcp add --help` command.
