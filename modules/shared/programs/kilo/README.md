# kilo

## 🤹 Installing agent skills

You can use the `skills` CLI([vercel-labs/skills](https://www.npmjs.com/package/skills)) or `gh skill` to install agent skills.

```bash
$ skills add https://github.com/github/awesome-copilot --agent kilo --skill documentation-writer
```

Or,

```bash
$ gh skill install --agent kilo github/awesome-copilot documentation-writer
```

## 🛠️ Setting up MCP servers

Run `kilocode mcp add` and follow the instructions to add MCP servers.
